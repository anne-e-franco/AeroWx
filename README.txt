Overview:
AeroWx is intended to be a one-stop graphical dashboard for aeronautical weather information. 
Will feature current conditions for the selected weather station (currently KBTL only).

Interface:
Clouds graphics which emulate actual cloud cover and ceiling height.
Weather conditions: snow/fog/showers/thunderstorms.
Wind 'compass' with a needle indicating wind direction and speed.


Data Sources:
ADDS textual data service http://weather.aero/tools/dataservices/textdataserver
